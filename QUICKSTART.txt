🚀 BƯỚC 2: CHẠY DEMO NHANH (2 PHÚT)
───────────────────────────────────────────────────────────────────────────

Option A: Python script
  $ python quick-demo-vit5.py

  Output expected:
    Input:  "tóm tắt: Trí tuệ nhân tạo..."
    Output: "AI là lĩnh vực nghiên cứu..."

Option B: Python interactive
  $ python
  >>> from transformers import AutoTokenizer, AutoModelForSeq2SeqLM
  >>> tokenizer = AutoTokenizer.from_pretrained("VietAI/vit5-base")
  >>> model = AutoModelForSeq2SeqLM.from_pretrained("VietAI/vit5-base")
  >>> text = "tóm tắt: Việt AI là tổ chức phi lợi nhuận. </s>"
  >>> inputs = tokenizer(text, return_tensors="pt")
  >>> outputs = model.generate(**inputs, max_length=50)
  >>> print(tokenizer.decode(outputs[0], skip_special_tokens=True))
  "Việt AI là tổ chức phi lợi nhuận..."

🎯 TẠI SAO CHỌN VIT5?

✓ State-of-the-art cho tiếng Việt
✓ Mở (MIT license)
✓ Dễ sử dụng với HuggingFace
✓ Nhiều finetuned variants có sẵn
✓ nhẹ (220M-770M params) → Chạy trên CPU/GPU phổ thông

🆚 VIT5 vs QWEN - NÊN DÙNG CÁI NÀO?

Task bạn cần                          Khuyến nghị
────────────────────────────────────────────────────────────
Dịch Anh-Việt, Việt-Anh              → ViT5 (envit5-base)
Tóm tắt văn bản                      → ViT5 (vietnews-summarization)
Hỏi đáp từ ngữ cảnh                  → ViT5 (fine-tune NER)
Chatbot hội thoại                    → Qwen
Sinh văn bản sáng tạo                → Qwen
Code generation                      → Qwen
Multitask (text + code)              → Qwen
Task có input-output rõ ràng         → ViT5
Task mở, cần sáng tạo                → Qwen

🐛 HAVE ISSUES?

Problem: Out of memory
→ Solution: Use FP16 or reduce batch size

Problem: Output doesn't match task
→ Solution: Add task prefix (tóm tắt:, dịch:, etc.)

Problem: Output too short
→ Solution: length_penalty=2.0, min_length=30

Problem: Slow inference
→ Solution: num_beams=1 (greedy), use FP16, batch size >1
