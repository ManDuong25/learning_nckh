## 💡 Why Notebooks?

1. **Interactive:** Chạy từng cell, xem output ngay
2. **Explainable:** Markdown cells giải thích từng bước

4. **Visual:** Tables, formatting, emoji
5. **Shareable:** Push lên GitHub, người khác có thể chạy
6. **Educational:** Perfect for learning (cells nhỏ, từng bước)

## 🎁 Bonus: Notebook Shortcuts

### Navigation
- `Shift+Enter` — Run cell, next cell
- `Ctrl+Enter` — Run cell, stay
- `Alt+Enter` — Run cell, insert below

### Edit
- `A` — Insert cell above (command mode)
- `B` — Insert cell below
- `D,D` — Delete cell
- `Z` — Undo delete

### Kernel
- `I,I` — Interrupt kernel
- `0,0` — Restart kernel
